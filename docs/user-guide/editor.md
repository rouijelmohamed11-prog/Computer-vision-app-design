# Core Editor

The VisionStudioAI Desktop Editor is a powerful environment designed for precision and flexibility.

## The Interface

The editor is divided into four main sections:

1.  **Top Bar**: Access AI Generation, Global Undo/Redo, and Export options.
2.  **Left Sidebar**: Contains the Toolbar, Asset Library, AI Suggestions, and Templates.
3.  **Central Canvas**: Your creative workspace where elements are placed and manipulated.
4.  **Right Inspector**: Detailed controls for element properties, layers, and color palettes.

## Working with Elements

You can add four types of elements to the canvas:

- **Images**: Drag and drop from your computer or select from the Asset Library.
- **Text**: Add highly customizable typography with support for various styles.
- **Shapes**: Primitive shapes (rectangles, circles) for layout and decoration.
- **Lines**: Vector lines for connections and borders.

### Manipulation

- **Transform**: Click and drag handles to resize or rotate elements.
- **Move**: Drag elements across the canvas.
- **Nudge**: Use arrow keys for pixel-perfect positioning (Hold `Shift` for 10px increments).
- **Z-Order**: Use the Layers panel to bring elements forward or send them to the back.

## View Controls

- **Zoom**: Use the scroll wheel or the zoom indicator in the top bar to scale your view.
- **Pan**: Hold the `Spacebar` and drag to move around a large canvas.
- **Reset View**: Click the "Reset View" button in the top bar to return to 100% zoom and center.
