# Advanced Editing Techniques

Master the VisionStudioAI canvas with these advanced manipulation tools.

## Precision Transformation

While dragging handles is fast, sometimes you need exact control.

### Using the Property Panel
1.  Select an element.
2.  In the **Transform** section, use the spin boxes for pixel-perfect X/Y coordinates and Dimensions.
3.  Use the **Rotation Slider** to set an exact angle from 0 to 359 degrees.

### Opacity Control
Every element has an **Opacity** slider in the Appearance section. This allows you to:
*   Create subtle watermarks.
*   Blend shapes for complex overlays.
*   Fade out background images.

## Layer Management

Understanding the Z-index is key to complex designs.

*   **Bring to Front**: Moves the element to the very top.
*   **Send to Back**: Moves it behind everything else.
*   **Incremental Move**: Use the Layers panel to move an element up or down one level at a time.

## Keyboard Shortcuts

Work faster with these built-in shortcuts:

| Action | Shortcut |
| :--- | :--- |
| **Undo** | `Ctrl + Z` |
| **Redo** | `Ctrl + Y` |
| **Delete** | `Del` or `Backspace` |
| **Nudge** | Arrow Keys |
| **Large Nudge** | `Shift + Arrow Keys` (10px) |
| **Pan Canvas** | `Spacebar + Drag` |
| **Zoom** | `Mouse Wheel` |
