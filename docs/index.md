# Welcome to VisionStudioAI Documentation

VisionStudioAI is a modular application integrating advanced NLP, vision processing, and frontend components to provide an AI-driven design environment.

## Project Overview

- **Core Engine:** Manages application state and execution (`core/`).
- **NLP Engine:** Handles natural language processing, intent identification, and response generation (`nlp/`).
- **User Interface:** Provides the graphical interface and interactive elements (`ui/`, `studio-frontend/`).
- **Vision Processing:** Tools for image analysis (`vision/`).
